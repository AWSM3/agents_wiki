# 0.1. Цели раздела

- **Last reviewed**: 2026-01-15
- **Уровень**: Basic

## Зачем эта страница

Цели раздела — сделать применение AI‑агентов **стандартизированным, безопасным и измеряемым**.

## Цели раздела (как "Definition of Why")

- **Ускорение SDLC**: разработка, рефакторинг, тесты, ревью, дебаг, документация, DevOps.
- **Обучение единым workflow**: чат‑боты для исследований + IDE/CLI‑агенты для имплементации.
- **Снижение рисков**: предотвращение утечек, prompt‑injection, неконтролируемой автономности, "вайб‑кодинга".
- **Повторяемость результатов**: меньше "как повезёт", больше "как в инструкции".

## Критичное допущение

Модель **не знает** ваш код и домен "по умолчанию". Качество резко растёт, когда вы:

- даёте **контекст‑пакет** (цель, ограничения, файлы, AC, логи/пример),
- требуете **план** перед изменениями,
- требуете **проверки** (tests/build/static),
- не нарушаете **политику данных**.
