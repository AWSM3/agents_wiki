# 10.4.L. Уровни владения навыками (L1–L3)

- **Last reviewed**: 2026-01-15
- **Уровень**: Basic

## Зачем эта страница

Понять “чему учить”, чтобы выросли качество и доверие к агентам.

## Компетенции (карта дефицитов)

1. **Промптинг по уровням E0–E4**  
   Симптомы: много итераций, расплывчатые запросы.  
   Практика: переписать 5 реальных задач в E2–E4.

2. **Контекст‑пакеты (Context Pack)**  
   Симптомы: “не учёл репо”, “сломал стиль”.  
   Практика: собрать Context Pack для 3 модулей.

3. **AI‑ревью и проверка**  
   Симптомы: принимают код “на веру”, ловят регресс поздно.  
   Практика: чек‑лист DoD + тест на регрессию.

4. **Работа с агентом (plan‑first)**  
   Симптомы: агент “колбасит во всём репо”.  
   Практика: план + границы + лимит на diff.

5. **Безопасность/секреты/политика данных**  
   Симптомы: в контексте попадают критичные части.  
   Практика: сценарии “как обезличить лог”, “как работать с D2/D3”.

6. **Модели и экономика**  
   Симптомы: “долго и дорого”, не знают когда переключаться.  
   Практика: 3 задачи → подобрать режим/модель.

## Уровни владения (L1–L3)

- **L1**: знает базовые шаблоны, уверенно делает E2.
- **L2**: делает E3, умеет Context Pack, стабильно проходит DoD.
- **L3**: делает E4, умеет автономность A2/A3 безопасно, наставляет других.
