# Tool Card: Kilo Code (open‑source, multi‑agents, parallel)

- **Last reviewed**: 2026-01-15
- **Уровень**: Advanced

## Что это

Open‑source платформа/агент с фокусом на многоагентность и параллельную работу (часто через git worktrees/ветки).

## Где запускается

- зависит от варианта: IDE/CLI (уточнить под вашу установку)

## Ключевые возможности

- параллельные агенты на разные подзадачи
- удобен для “одна задача → несколько агентов → человек объединяет”

## Как задавать правила/инструкции

- каждому агенту — узкий scope + DoD + запреты
- общий source‑of‑truth: Spec/ADR/TestPlan
- запрет интеграционных действий без человека

## Риски/ограничения

- конфликты и “разъезд” решений между агентами
- повышенный риск утечек, если агентам дать широкий контекст

## Мини‑рецепты

- агент A: план/арх‑варианты
- агент B: код‑патч
- агент C: тесты/verify
