# Tool Card: Roo Code (VS Code, open‑source агент)

- **Last reviewed**: 2026-01-15
- **Уровень**: Advanced

## Что это

Open‑source агент в VS Code: умеет multi‑file edits, режимы и (часто) MCP‑интеграции.

## Где запускается

- VS Code (локально)

## Ключевые возможности

- настраиваемые режимы (под разные workflow)
- гибкая настройка подключаемого контекста
- интеграции через MCP (если включаете)

## Как задавать правила/инструкции

- AGENTS.md как общий стандарт
- режимы/настройки контекста должны исключать секретные папки
- для MCP: allowlist и least privilege

## Риски/ограничения

- open‑source плагины/интеграции = дополнительный риск supply chain
- неправильная настройка контекста может “подхватить” чувствительные файлы

## Мини‑рецепты

- “Сделай plan‑first refactor в пределах папки X, без новых зависимостей, с тест‑планом”
