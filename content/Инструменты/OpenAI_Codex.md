# Tool Card: OpenAI Codex / Codex‑style CLI (coding agent)

- **Last reviewed**: 2026-01-15
- **Уровень**: Advanced (рекомендации) / Basic (понимание)

## Что это

Класс инструментов "coding agent" в CLI: агент работает с репозиторием и приносит патчи/изменения, удобен для процедурных тех‑задач.

## Где запускается

- терминал (локально или в изолированном окружении)

## Ключевые возможности

- план → патч → (по подтверждению) verify
- хорошо подходит для "agent‑runner" сценариев (CI) при наличии guardrails

## Как задавать правила/инструкции

- AGENTS.md + шаблоны (Spec/ADR/TestPlan)
- запрет сетевых действий/запуска команд без approve
- жёсткая политика данных (D2/D3 — только локально)

## Риски/ограничения

- терминал и сеть повышают риск эксфильтрации
- без ограничений может затронуть критичные зоны

## Мини‑рецепты

- "Сделай план + patch; затем предложи команды verify и дождись approve"
