# 3.3. Tool Cards — карточки инструментов (единый формат)

- **Last reviewed**: 2026-01-15
- **Уровень**: Basic → Advanced

## Зачем эта страница

Tool Card — “паспорт” инструмента: что он умеет, как безопасно использовать, как задавать правила, какие рецепты (mini‑playbooks) есть.

## Единый формат Tool Card

- что это (1 абзац)
- где запускается (IDE/CLI/облако/CI)
- ключевые возможности
- как задавать правила/инструкции (AGENTS.md / rules / commands / CLAUDE.md)
- privacy/retention и политика данных
- риски/ограничения
- мини‑рецепты (2–3)
- ссылки на оф. доки
