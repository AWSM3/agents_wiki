# Tool Card: GitHub Copilot (IDE + agent mode)

- **Last reviewed**: 2026-01-15
- **Уровень**: Basic → Advanced

## Что это

IDE‑инструмент (autocomplete + chat + agent mode). Важно различать “agent mode в IDE” и автономных PR/CI‑агентов (если они используются).

## Где запускается

- IDE (локально)
- (опционально) PR/CI‑агенты — в окружении CI (по отдельной политике)

## Ключевые возможности

- autocomplete и чат
- agent‑подход: выбор файлов/правки/итерации (зависит от режима)

## Как задавать правила/инструкции

- `.github/copilot-instructions.md` (репозиторные инструкции)
- path‑specific `.instructions.md` (если используете раздельные зоны)
- общий source‑of‑truth: `AGENTS.md` (см. [10.3](../Приложения/10_03_Templates_Index.md))

## Privacy/retention (что проверить)

- условия плана вашей организации (Business/Enterprise)
- ретеншн prompts/outputs и доступ к логам
- запреты по данным D2/D3

## Риски/ограничения

- та же группа рисков, что и у IDE‑агентов (prompt injection, scope creep)
- дополнительный риск, если включены внешние интеграции

## Мини‑рецепты

- “Сгенерируй тест на регрессию + минимальный фикс” (E3/E4)
- “Составь план рефакторинга с ограничением на scope”
