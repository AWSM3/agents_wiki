# 3.5. MCP (Model Context Protocol): что это, где применять, безопасность

- **Last reviewed**: 2026-01-15
- **Уровень**: Advanced (core) / Basic (понимание)

## Зачем эта страница

MCP — один из способов стандартизировать подключение ассистента к инструментам/данным. Важно понимать, где он даёт ценность и какие риски создаёт.

## Где применяем

- доступ к внутренним докам (через безопасный прокси)
- доступ к CI/артефактам
- чтение метрик/логов/трейсов (read‑only)

## Базовые правила безопасности

- allowlist серверов
- least privilege / scopes
- tool approval (особенно write‑действия)
- аудит вызовов tools
