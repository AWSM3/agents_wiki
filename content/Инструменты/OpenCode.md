# Tool Card: OpenCode (terminal/TUI, open‑source)

- **Last reviewed**: 2026-01-15
- **Уровень**: Advanced

## Что это

Open‑source terminal/TUI агент, часто поддерживает multiple providers. Подходит как CLI‑агент для многошаговых задач.

## Где запускается

- терминал (локально/на выделенной машине)

## Ключевые возможности

- итеративные изменения с опорой на репозиторий
- удобно для скриптов/CI/миграций

## Как задавать правила/инструкции

- AGENTS.md / project docs как источник истины
- запрет auto‑run и интеграционных действий
- ограничение сети/интеграций по политике

## Риски/ограничения

- терминал = повышенный риск (см. threat model)
- провайдер‑маршрутизация должна соответствовать политике данных
