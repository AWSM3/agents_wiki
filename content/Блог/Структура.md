# Структура проекта

## Директории

### `content/`
Здесь хранятся все Markdown файлы с контентом. Поддерживаются вложенные папки.

```
content/
  blog/
    welcome.md
    structure.md
  docs/
    guide.md
    api.md
```

### `src/`

- **`pages/`** — компоненты для отдельных роутов (Home, Post)
- **`utils/`** — утилиты для загрузки и парсинга контента
- **`main.ts`** — entry point приложения
- **`router.ts`** — конфиг Vue Router

### `dist/`

Сгенерированная production статика (создаётся после `npm run build`).

## Как добавить новый пост?

1. Создайте файл в `content/blog/` (или в другой папке):
   ```bash
   content/blog/my-article.md
   ```

2. Добавьте YAML frontmatter и контент:
   ```markdown
   ---
   title: "Мой статья"
   date: "2025-12-25"
   ---

   # Содержимое...
   ```

3. Запустите `npm run dev` — пост появится на главной!

## Маршруты

Структура `content/` автоматически транслируется на клиентские маршруты:

- `content/blog/welcome.md` → `/post/blog/welcome`
- `content/docs/guide.md` → `/post/docs/guide`
